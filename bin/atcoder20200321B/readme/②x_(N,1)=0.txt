①x_(N,1)=1.txt による考察の後、x_(N,1) が 1 ではないことを前提とする。


■仮定
   a_k (1 <= k <= N) において,
   a_k = 2 が存在すれば、x_(N,1)=０

■証明

 [1] a_k = 2 (1 <= k) のみの数列である時
  | a_k - a_k+1 | = 0 であるため、 x_(N,1) = 0
  
 [2] a_k = 2 (1 <= k) のみではない時
  いずれかの| a_k - a_k+1 | = 1 となり、 x_(N,1) = 2 ではない。
  ①より x_(N,1) が 1 ではないため、x_(N,1) = 0
